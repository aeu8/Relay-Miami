# Relay Miami Website

This is a ready-to-deploy Next.js website for Relay Miami.

## What this site does
- Presents the Relay Miami concept
- Shows sample secondhand furniture listings
- Includes prototype forms for:
  - Seller listings
  - Buyer item requests
  - Delivery quote requests for outside listings

## Important
The website will go live once deployed, but the forms are still demo forms.
They currently display a success message on the page rather than sending submissions to you.

The next build step is connecting the forms to:
- email notifications,
- Google Sheets / Airtable / Supabase,
- and eventually an admin dashboard.

## How to deploy
The simplest path:
1. Upload this folder to GitHub.
2. Connect the GitHub repo to Vercel.
3. Click Deploy.

Vercel will detect Next.js automatically.
